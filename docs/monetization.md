## 배치 파이프라인 (경기 시간 기준 동적)

```
매시간 cron 실행 → 경기 시작 시간 기준 자동 판단:
  경기 4시간 전 ±30분 → Phase 1 (선발투수 기반)   → Free 제공
  경기 1시간 전 ±30분 → Phase 2 (라인업 반영)     → Basic/Pro 제공
  Pro만               → 수동 재분석 (reanalyze)   → 실시간 LLM
```

예시 (평일 18:30 경기 / 주말 14:00 경기):
| | 평일 18:30 | 주말 14:00 |
|---|---|---|
| Phase 1 | 14:30 | 10:00 |
| Phase 2 | 17:30 | 13:00 |

일 고정 비용: 5경기 × 2회 × ~₩150 = ₩1,500/일 ≈ ₩45,000/월

## 법적 필수 사항

1. **사업자 등록** (개인사업자, 업종: 63120)
2. **통신판매업 신고** (관할 구청, 유료 서비스 제공 전 필수)
3. 이용약관 업데이트 (결제/환불/자동갱신)
4. 개인정보처리방침 업데이트 (결제 정보 수집)
5. 면책 고지 유지 (도박 관련 차단)
6. 자동갱신 7일 전 사전 알림 의무
7. 미성년자 결제 제한

## 절대 금지 표현

"적중", "픽", "배팅", "수익률", "보장", "토토" → 도박법 저촉 리스크

## 구현 로드맵

### Phase 1: MVP (3주)
- 토스페이먼츠 가맹점 가입
- 단건 월간 결제 (자동갱신 없이)
- 프론트: 결제 여부 → 콘텐츠 노출 제어

### Phase 2: 정식 구독 (4-6주)
- 빌링키 정기결제
- 구독 관리 DB
- 자동갱신 + 해지 UI

### Phase 3: 고도화 (2-3개월)
- 쿠폰/프로모션
- 카카오 알림톡
- B2B API
